# Privacy Policy

## Death & Taxes Scanner

Last updated: March 4, 2026

### Data Collection

This extension does not collect, store, or transmit any personal data. Period.

### What the extension does

The extension reads NFT token IDs from the OpenSea collection page and fetches their on-chain tax status from the Ethereum blockchain through a public API. This data includes tax payment status, audit timers, insurance status, and bribe balances. All of this is public blockchain data, not user data.

### What the extension does NOT do

- Does not collect your name, email, wallet address, or any personal information
- Does not track your browsing history or activity
- Does not use cookies or local storage
- Does not collect analytics or telemetry
- Does not transmit any data about you to any server
- Does not run on any page other than the Death & Taxes collection on OpenSea

### Permissions

- **Host permission (proxy-death.vercel.app):** Used to fetch public blockchain data for NFT tax status display
- **Content script (opensea.io/collection/deathandtaxes):** Used to inject badge overlays onto NFT cards on the collection page only

### Open Source

The full source code is available at https://github.com/0xcaptainy/deathandtaxes

### Contact

For questions about this privacy policy, open an issue on the GitHub repository.
